# WINBOX Landing Page

Premium AI-powered digital marketing landing page built with:

- Next.js 16 (App Router)
- TypeScript (strict)
- Tailwind CSS v4
- Shadcn UI style components
- Framer Motion animations
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm start
```
